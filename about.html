About the Project

AllLingo.F1F4 is an AI-powered real-time language translator designed to break language barriers across the globe. Our project provides the ability to translate text in over 150+ languages. Users can interact with the application through typed text, voice input, or even by uploading images containing text — similar to Google Lens.

Our system also supports intelligent features such as text summarization using Transformer models, optical character recognition (OCR) using Tesseract, and voice output using Google Text-to-Speech (gTTS). This allows a user to summarize, translate, and listen to content from various input formats in real time.

The project is developed as part of our academic journey with the goal of creating a powerful, easy-to-use, and accessible translation tool using modern technologies in AI and Natural Language Processing.
